# Inscripedia

> Entire books inscribed on Bitcoin as recursive ordinals, read in a 3D viewer.

![inscription 33633842 rendering live from Bitcoin](media/inscripedia-onchain.png)
*Rendering live from the chain via ordinals.com — inscription 33633842*

A Three.js book viewer that reads EPUBs stored as recursive ordinal inscriptions — an entire book, on Bitcoin, readable in 3D in the browser. Shipped with a web2 EPUB reader, an ordinal generator, and a "gift wrap" component. Built and handed over as infrastructure for the Inscripedia project.

## highlights

- EPUB → zip.js → recursive inscriptions → 3D page-turning viewer
- On-chain media as usable product, not provenance JPEGs
- Delivered as a client handover
