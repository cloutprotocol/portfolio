# leisure.delivery

> On-demand delivery platform built for **Lil Yachty**. CTO & co-founder.

End-to-end delivery platform: native Swift/SwiftUI iOS app with Apple Maps integration, dispatch system, payment and order workflows, and **Dynamic Island widgets + Live Activities** tracking orders in real time over WebSockets. Convex backend, Dockerized Node API deployed via Coolify, plus a web client.
