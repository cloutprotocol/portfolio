# riftrunner

> A browser MOBA built from scratch in Three.js. No engine.

Champions, abilities UI, towers, projectiles, minimap, demo mode, and custom anamorphic post-processing — raw Three.js + Vite, ES modules. No engine, no asset store: game loop, netcode-shaped architecture, and post pipeline written from scratch.

The same graphics fluency that runs through the [zatoshi](#/zatoshi) dither shaders, [purple.trade](#/purple-trade)'s R3F layer, and the [BORNSINNER](#/bornsinner) mint site — pushed to its logical extreme.
