# BORNSINNER

> A complete Bitcoin Ordinals collection launch stack — 3D mint site to art generator.

End-to-end launch infrastructure across 7 repos: a 3D mint site (React Three Fiber + postprocessing, Unisat wallet SDK, bitcoinjs-lib), a 404-style mint variant, API backend, countdown site, and generator.
