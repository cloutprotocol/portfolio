# recursive-doom

> DOOM running in the browser through recursive Bitcoin ordinal inscriptions. June 2023 — weeks after recursion landed.

![recursive-doom](media/recursive-doom-repo.png)

A proof-of-concept that loads DOOM via an inscribed js-dos API (`/content/f70eb...i0`) with the game file on IPFS. The README solicited ~0.3 BTC to inscribe the game fully on-chain for v2.

Companion utility: **recursive-ordinal-loader** — a JS loader for brotli-compressed recursive files that shrinks on-chain inscription size for recursive art and apps.
