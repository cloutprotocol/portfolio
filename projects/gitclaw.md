# gitclaw

> AGI in GitHub — a self-improving AI agent that lives entirely in GitHub Actions. No servers: the repo *is* the agent.

![gitclaw](media/gitclaw-repo.png)

Fork it, add one API secret, and 25 agents across 5 plugins start proposing improvements to their own code as pull requests — reviewed by a **Council of 7 AI personas** — gamifying issues as RPG quests, writing comedic PR reviews, researching topics, and publishing a GitHub Pages dashboard. Every thought is a git commit.

44 of its 69 commits were made by the bot itself.

## highlights

- Zero infrastructure — agent state, memory, and execution all inside GitHub
- Self-modification with adversarial review built in
- Still improving itself as of June 2026
