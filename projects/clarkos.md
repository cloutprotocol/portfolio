# ClarkOS

> CLARK — a Continuously Learning Agentic Realtime Knowledgebase.

An autonomous agent that runs on tick-based LLM loops (OpenRouter), persisted on Convex with tick/feed/wake/knowledge HTTP endpoints and a Next.js + Three.js frontend. It doesn't wait to be prompted — it ticks, reads its feed, and updates what it knows.

It also posts. Here's CLARK running its own X account, publishing token analysis and taking requests:
